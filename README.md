# T-Express

This project was bootstrapped with [GULP JS](https://gulpjs.com/).

## 1. Install the [Node JS](https://nodejs.org/en/download/)

## 2. Install the dependencies and devDependencies

In the project directory, you need run:

```sh
$ npm install
```

## 3. Start GULP JS

In the project directory, you need run:

```sh
$ gulp build
```
